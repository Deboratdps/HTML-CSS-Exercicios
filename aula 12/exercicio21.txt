1. É verdade que todo arquivo CSS precisa estar dentreo de outra pasta para que o navegador possa renderizar os
estilos das propriedades CSS?
R: Não é verdade, pode ter um arquivo de estilo na pasta raiz. É legal para manter numa pasta específica para melhor
utilização.

2. É verdade que todo elemento HTML possui por padrão elementos filhos?
R: Não é verdade, alguns elementos em HTML possam sim ter um ou outro elemento filhoe, mas nem todos os elementos
HTML terão elementos filhos.
